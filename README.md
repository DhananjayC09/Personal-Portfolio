# Portfolio

Responsive personal portfolio website built using HTML, CSS, and JavaScript. Showcases projects, skills, and contact info with modern UI features like smooth scrolling, glassmorphism cards, and scroll animations.

## Overview 

This is a **responsive personal portfolio website** built using **HTML, CSS, and JavaScript**.  
It showcases my projects, skills, and contact information with a clean, modern UI. 

## Features 

- Smooth scrolling navigation 
- Glassmorphism design cards 
- Responsive layout for all screen sizes 
- Contact section with email link and resume download 
- Project showcase section with images
